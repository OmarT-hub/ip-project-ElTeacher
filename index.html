Introduction to Html

  1. HTML stands for HyperText Markup Language
  2. It is the standard language used to create and structure web pages. HTML describes the content of a webpage using elements called tags, such as headings, paragraphs, images, and links.
  3. HTML is not a programming language — it is a markup language, which means it tells the browser how to display text, images, and other content on the screen.
  4. Every HTML page is made of a collection of elements, and each element has an opening tag, content, and a closing tag.

Example

<p>This is a paragraph in HTML.</p>

Basic Structure of an HTML Page

  1. Every HTML page follows a basic structure that tells the browser how to read and display the content. This structure consists of several essential elements that define the document type, the head section, and the body section.

Example

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>

  <body>
    <h1>Welcome to My Website</h1>
    <p>This is a simple HTML page.</p>
  </body>
</html>

<!DOCTYPE html> – Declares the document type and version of HTML.

<html> – The root element that contains the entire HTML document.

<head> – Contains meta-information about the document, like the title, character set, and links to CSS or JavaScript files.

<title> – Sets the title of the webpage, which appears in the browser tab.

<body> – Contains all the content that is displayed on the webpage, like headings, paragraphs, images, and links.

<h1> to <h6> – Heading elements, with <h1> being the largest.

<p> – Paragraph element for text content.

Common HTML Tags

  1. Headings: <h1> to <h6> for titles and headings.
  2. Paragraphs: <p> for normal text content.
  3. Links: <a href="URL"> for hyperlinks.
  4. Images: <img src="URL" alt="Description"> for pictures.
  5. Lists: <ul> or <ol> with <li> for list items.
  6. Forms: <form> with <input>, <textarea>, <button> for user input.
  7. Divs & Spans: <div> for block elements, <span> for inline elements.

Example

<h1>This is a heading</h1>
<p>This is a paragraph</p>
<a href="https://www.example.com">Visit Example</a>
<img src="image.jpg" alt="Example Image">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Headings & Paragraphs

  1. Headings are defined with <h1> to <h6> tags. <h1> is the largest heading and <h6> is the smallest.
  2. Paragraphs are defined with the <p> tag. They are used to display normal text content on the page.
  3. Using headings and paragraphs properly helps organize content and improve readability.

Example

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>

<p>This is a paragraph of text.</p>
<p>Another paragraph example.</p>

Links and Images

  1. Links are created using the <a> tag with the href attribute to specify the URL.
  2. Images are added using the <img> tag with src for the image URL and alt for alternative text.
  3. Proper use of links and images improves navigation and visual content on a webpage.

Example

<a href="https://www.example.com">Visit Example</a>

<img src="https://via.placeholder.com/150" alt="Placeholder Image">

Lists (ul, ol, li)

  1. Unordered lists (<ul>) are used for lists with bullet points.
  2. Ordered lists (<ol>) are used for numbered lists.
  3. Each item in a list is defined using the <li> tag.
  4. Lists help organize information and improve readability on webpages.

Example

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

Tables Basics

  1. Tables are used to display data in rows and columns.
  2. The <table> tag defines a table.
  3. <tr> defines a table row, <th> defines a table header, and <td> defines a table cell.
  4. Tables help organize structured data in a readable format on a webpage.

Example

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>Omar</td>
    <td>25</td>
    <td>Cairo</td>
  </tr>
  <tr>
    <td>Ahmed</td>
    <td>22</td>
    <td>Alexandria</td>
  </tr>
</table>
Name Age City
Omar 25 Cairo
Ahmed 22 Alexandria

Forms and Inputs

  1. Forms are used to collect user input on a webpage.
  2. The <form> tag defines a form, and input fields are added using <input>, <textarea>, <select>, and <button> tags.
  3. Each input type can be text, email, password, checkbox, radio, or submit, depending on the required user input.
  4. Forms help users interact with the website, like signing up, logging in, or submitting data.

Example

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" placeholder="Enter your name">

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" placeholder="Enter your email">

  <button type="submit">Submit</button>
</form>

Divs and Spans

  1. The <div> tag is a block-level container used to group other HTML elements. It helps organize the layout of a webpage.
  2. The <span> tag is an inline container used to style or group small pieces of content within other elements, like text inside a paragraph.
  3. Using divs and spans allows you to apply CSS styles and structure content efficiently.

Example

<div class="container">
  <h2>This is a heading inside a div</h2>
  <p>This is a paragraph with a <span style="color: red;">highlighted text</span> inside.</p>
</div>

This is a heading inside a div

This is a paragraph with a highlighted text inside.

Semantic HTML Elements

  1. Semantic HTML elements clearly describe their meaning and purpose in the code, which improves accessibility and SEO.
  2. Examples include:
    • <header> – Represents the header of a page or section.
    • <nav> – Defines a navigation menu.
    • <main> – Contains the main content of the page.
    • <section> – Groups related content together.
    • <article> – Represents an independent, self-contained piece of content.
    • <aside> – Contains content related to the main content, like sidebars.
    • <footer> – Represents the footer of a page or section.
  3. Using semantic elements helps browsers, search engines, and developers understand the structure of your webpage.

Example

<header>
  <h1>Website Header</h1>
</header>

<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
  </ul>
</nav>

<main>
  <article>
    <h2>Article Title</h2>
    <p>This is an article.</p>
  </article>
</main>

<aside>
  <p>Related content goes here.</p>
</aside>

<footer>
  <p>Footer information</p>
</footer>

HTML Attributes

  1. HTML attributes provide additional information about an element.
  2. Attributes are always written inside the opening tag of an element.
  3. Common attributes include:
    • id – Unique identifier for an element.
    • class – Used to apply CSS styles or select elements with JavaScript.
    • href – URL for a link (<a> tag).
    • src – Source of an image or media file (<img> tag).
    • alt – Alternative text for images (<img> tag).
    • title – Provides extra information when hovering over an element.
    • placeholder – Placeholder text in input fields (<input> tag).
  4. Attributes help customize and control the behavior and appearance of HTML elements.

Example

<a href="https://www.example.com" title="Go to Example">Visit Example</a>

<img src="image.jpg" alt="Example Image" width="200">

<input type="text" placeholder="Enter your name" class="input-field">
Visit Example

Comments in HTML

  1. Comments are notes in the HTML code that are not displayed in the browser.
  2. They are useful for explaining code, leaving reminders, or temporarily disabling code.
  3. Comments start with <!-- and end with -->.
  4. Browsers ignore comments, so they have no effect on the page display.

Example

<!-- This is a comment in HTML -->

<p>This paragraph will be displayed.</p>

<!-- <p>This paragraph will NOT be displayed.</p> -->

This paragraph will be displayed.

Example HTML Code

  1. Here is a complete example of a simple HTML page combining many of the elements we've learned:

Example

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My First HTML Page</title>
</head>
<body>

<!-- Header Section -->
<header>
  <h1>Welcome to My Website</h1>
</header>

<!-- Navigation -->
<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

<!-- Main Content -->
<main>
  <section>
    <h2>Introduction</h2>
    <p>This is a paragraph explaining the website.</p>
  </section>

  <section>
    <h2>List Example</h2>
    <ul>
      <li>Item One</li>
      <li>Item Two</li>
      <li>Item Three</li>
    </ul>
  </section>

  <section>
    <h2>Form Example</h2>
    <form>
      <input type="text" placeholder="Enter your name">
      <button type="submit">Submit</button>
    </form>
  </section>
</main>

<!-- Footer Section -->
<footer>
  <p>Copyright © 2025</p>
</footer>

</body>
</html>
